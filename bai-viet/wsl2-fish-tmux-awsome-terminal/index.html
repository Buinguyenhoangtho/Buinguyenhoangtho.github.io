CÀI ĐẶT WSL2 trên Windows 10

Yêu cầu

  Window 10: Version 1903 or higher, with Build 18362 or higher.

Mở powershell (run as Admin) và chạy lệnh:

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

Khởi động lại máy tính

  • Tải distro Ubuntu trên Windows Store và đăng kí user

Cài đặt fish-shell

Fish shell aka (Friendly interactive shell) thật sự là một sự thay thế đáng giá cho cái bash mặc định “củ chuối” của Linux Cụ thể, khi sử dụng fish. Bạn sẽ được những tính năng sau

  • Hiện gợi ý các câu lệnh đã gõ ( như khi search website)
  • Nhẹ - reponse nhanh
  • Có thể tuỳ chỉnh giao diện nhanh thông qua giao diện web (fish_config)

Tại cửa sổ wsl, gõ tiếp những lệnh sau

## cài đặt fish-shell
apt-get update
apt-get install fish
  • fish-shell mặc định, nhập
chsh -s `which fish`

Khởi động lại WSL

Cài đặt plugins omf ( oh-my-fish) và theme

###oh-my-fish

Một khi sử dụng fish-shell thì bạn nên cài đặt thêm framework oh-my-fish. Nó cung cấp sẵn cho bạn nhiều theme hay plugin hay ho để bạn có thể “nghịch”

Gõ hoặc paste dòng lệnh dưới đây để cài đặt

curl -L https://get.oh-my.fish | fish

Mình sẽ giới thiệu sơ qua một vài dòng lệnh cơ bản của omf

omf list <lấy danh sách plugins/theme>
omf install <tên-theme/tên-plugins> # cài đặt theme/plugins
omf remove <tên theme/plugin> # xoá theme/plugins

Tmux:

Ảnh tham khảo

Định nghĩa trên trang https://tmux.github.io/ có nói rằng: Tmux là một Terminal Multiplexer - “bộ ghép kênh”. Nó cho phép bạn chuyển qua lại giữa các chương trình độc lập ngay trên một terminal, tách các chương trình ra một terminal riêng mà vẫn giữ được trạng thái hoạt động của chúng. Và còn làm được nhiều việc hơn thế nữa… Viblo

Để cài đặt tmux, sử dụng lệnh

sudo apt-get update 
sudo apt-get install tmux

Cài đặt xong tmux, các bạn CẨN cài đặt thêm plugins tmux-zen của oh-my-fish để tmux chạy mặc định trên fish-shell :

omf install tmux-zen

Config Tmux

Config này mình tham khảo tại đây. Để sử dụng chúng , các bạn gõ sudo nano ~/.tmux.conf và paste config. Nhớ lưu và khởi động lại fish-shell

# remap prefix from 'C-b' to 'C-a'
unbind C-b
set-option -g prefix C-a
bind-key C-a send-prefix

# split panes using | and -
# split panes using | and -
bind | split-window -h 
bind - split-window -v
unbind '"'
unbind %

# switch panes using Alt-arrow without prefix
bind -n C-Left select-pane -L
bind -n C-Right select-pane -R
bind -n C-Up select-pane -U
bind -n C-Down select-pane -D

#mouse
set -g mouse on
# remove window bind with control- w 
bind -n C-w kill-pane

# kill sessions 
bind -n C-S-w kill-session
# rightbar
set -g status-right ''

#tmp
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
# Other examples:
# set -g @plugin 'github_username/plugin_name'
# set -g @plugin 'git@github.com:user/plugin'
# set -g @plugin 'git@bitbucket.com:user/plugin'
# Initialize TMUX plugin manager (keep this line at the very bottom of tmux.conf)
run '~/.tmux/plugins/tpm/tpm'
setw -g window-status-current-format "#{pane_current_path}"  
# set interval 
set -g status-interval 1
#clock

set -g status-right '%d/%m - %H:%M:%S '
